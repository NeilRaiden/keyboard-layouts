# Keyboard Layout Writer+ (QWERTY)

This is a standard US English QWERTY keyboard layout, with added common punctuation marks and common symbols.

This layout is identical to **Writer (QWERTY)** but with additional ability to type IPA symbols for **English phonemes**.

>Note:  
>This layout is for someone who needs to type English phonemes only **occasionally**.  
>If you need to type whole words in phonemic English, then please try the [English (Phonemic)](Phonemic.md) layout from this project.

The IPA symbols can be entered with **AltGr** (right Alt) and **AltGr+Shift**, see the tables below.

### Spaces:

| Layer | Unicode | Whitespace name |
|:----- |:------- |:--------------- |
| Unshifted | U+0020 | regular space |
| Shifted | U+0020 | regular space |
| AltGr | U+2003 | Em-space - width equal to font height |
| AltGr+Shift | U+200b | zero-width space |
