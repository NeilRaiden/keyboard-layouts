English QWERTY

Standard QWERTY layout created in early 1870s for first typewriters.

English Phonemic

With this layout you can type in English with IPA symbols.
The Unshifted and Shifted layers are based on Shavian QWERTY layout.
The AltGr and AltGr+Shift layers are based on Writer+ QWERTY.

Writer QWERTY

Layout devised by Neil Raiden âŸ®â˜ºâŸ¯ to type frequently used symbols
that are not avilable on the standard QWERTY layout.

Writer+ QWERTY

This layout is similar to Writer QWERTY with added IPA symbols for English phonemes.
It is intended for only occasional typing.

Writer+ ASERTH

This layout is based on QWERTY-Flip-Twist by Nick Gravgaard
The AltGr layers are similar to Writer+.

Shavian QWERTY

This layout is based on Keyman Shaw QWERTY by Ed Greville; License: MIT.

Shavian+ QWERTY

Similar to Shaw QWERTY but extended with AltGr layers for extra punctuation marks.

Shavian Imperial

Based on the layout of the Imperial Model 6 typewriter.
The uppercase layer was moved to the AltGr. And Shifter layer contains lowercase Latin symbols. Also, some extra punctuation marks are added to AltGr+Shift.

Shavian 2-Layer

Based on Keyman Shaw (2Layer) by Haley Halcyon; License: MIT.
