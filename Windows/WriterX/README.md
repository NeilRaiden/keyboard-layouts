# Keyboard Layout (WriterX)

**Keyboard layout for entering common punctuation marks instead of using Alt-codes.**

**Added ability to enter English phonemes**

Required language pack: **English (United States)**

This is a standard US English QWERTY keyboard layout, with added common punctuation marks and common symbols.

The extra symbols can be entered with **AltGr** (right Alt) and **AltGr+Shift**, see the tables below.

## AltGr:

```
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬──────┐

├───┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬────┤
│ tab │ … │ ⋯ │ é │ ® │ ™ │ √ │ ↑ │ ɪ │ ɒ │ ℗ │ ⟨ │ ⟩ │  • │
├─────┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴────┤
│ caps  │ “ │ ” │ ↓ │ ° │ ɛ │ ‐ │ ɜ │ « │ » │ × │ § │  ent │

```
